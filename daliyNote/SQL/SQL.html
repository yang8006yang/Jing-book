Introduction

Database(資料庫) != DBMS(資料庫管理系統)

DBMS是用來操作和管理資料庫的軟體

  • Creat /Defining ：新增 指明資料結構、類型、限制
  • Consturcting ：建立 輸入並儲存
  • Read +Update +Delete/Manipulating： 讀取、查詢、更新、刪除
  • Sharing ： 多個使用者同時存取

Start up

使用phpmyadmin為例


  • Step.1 : 建立資料庫

    • 左側欄 按桶子新增資料庫 > 編碼選 [utf_general]系列 (萬國碼)

  • Step.2 : 建立資料表

    • 2.1 認識資料型態

    一些常用的
    種類 資料型態 說明
    整數 INT 4byte
    整數 TINYINT 1byte
    數字很小 或 需要使用布林值可用
    實數 (小數點) FLOAT p 小於等於 24
    日期 DATE 格式 YYYY-MM-DD
    日期時間 TIMESTAMP 格式 YYYY-MM-DD HH:MM:SS
    字串 VARCHAR 可變字串長度
    字串 CHAR 固定字串長度
    布林值 BOOLEAN false = 0

    2.2 輸入名稱、類型...ect.

    一些事項
    資料表命名 用複數+底線 ex. high_school_students
    PRIMARY 主鍵 資料不會重複的欄位，以做為不同資料表間關聯欄位
    *設主見才有edit bar
    A I auto increment
    關聯資料表 : 透過相同的欄位將不同的表關聯(一對一、一對多、多對多)
  • Step.3 : 新增資料
