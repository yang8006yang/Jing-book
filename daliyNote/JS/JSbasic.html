認識JavaScript

一種前端的腳本語言，可以與使用者行為互動

  • 可以改HTML的內容、屬性、樣式(CSS)
  • 偵測使用者行為(點擊、滾動...)
  • 要包在 script tag 內
  • 大小寫有差
  • 乾淨的JS Code怎麼寫

變數

宣告變數與其Scope
  • var : 看宣告位置決定是全域或區塊。(在function外宣告是全域)
  • let : 區塊變數，以{}區分，在同一{}內無法同名重複宣告
  • const : 常數，其他同let
  • 沒有以上直接賦值 : 全域屬性 (小心使用)
    • 全域變數=>window(全域物件)的屬性
  • 差異
    var let const
    scope作用域 在function上，var還能在外層取到值 在block上{}，作用域只存在{}內 在block上{}
    全域 O (會在window上) X X
    重複宣告 O X X
    hoisting 在宣告前使用，會hoisting(提升)，得到undefined 有 hoisting 但有暫時性死區(TDZ)則出現ReferenceError
    建議變數盡量在scope的最上面宣告完成後再使用

    函式也可提升 : 與變數提升的差異在於 變數提升只有宣告被提升，而函式提升是{}內都被提升
    let/const 的 TDZ : 如果在宣告變數之前使用變數，這個變數就是存在「暫時死區」中無法存取！

資料型態

JS是個"弱型別"的語言

  • 數值Number : 可以是整數或浮點數

    • 強制取小數點位數函式：(變數).toFixed(指定位數)

  • 字串String
  • 布林Boolean
  • Undefined : 尚未被賦值
  • Null : 空值
  • Object : 上述以外的 ex.Array、Date
  • Symbol
    • 常用轉換型別的函數
    parseInt() 將字串轉為整數 從最左邊開始轉換，無法轉換則回傳NaN
    parseFloat() 將字串轉為浮點數
    Number() 將字串/物件轉為數值 無法轉換則回傳NaN
    typeof 傳回資料型別 typeof(XX) / typeof XX

運算子

    指定運算子 = += -= *= /= %= a+=3; (a=a+3)
    算數運算子 + - * / % ++ -- 加減乘除餘數
    字串運算子 + 組合字串
    布林運算式 && (x > 10)&&(y > 15) => ture/false
    0、""、false、null、undefined、NaN 都是 false
    比較運算子 == != > >= < <= === 是完全相等(資料型態也要相同)
    邏輯運算子 && || !
    a++ v.s. ++a a++ : 做運算/賦值完在加 ++a : 先加在做運算/賦值

流程控制-邏輯判斷式

  • if(...){}else{}
  • 三元運算子：條件式?成立時:不成立時
  • switch...case
    • switch(變數/運算式){
    case A:
    程式敘述;
    break;
    case B:
    程式敘述;
    break;
    .
    .
    .
    default:
    程式敘述;
}
  • 短路求值 :

    短路求值是一種邏輯運算子的求值策略。只有當第一個運算數的值無法確定邏輯運算的結果時，才對第二個運算數進行求值。(form wiki)

    || 回傳第一個 Boolean() 判斷是 true 的值
    && 回傳第一個 Boolean() 判斷是 false 的值，都是ture回傳最後一個
    ?? (空值合併運算子)回傳第一個非 null 和非 undefined 的值，都沒有回傳最後一個

Array 陣列

JavaScript 的陣列有兩種，一種是 [] 包起來的 Array，另一種是 {} 包起來的 物件

  • Array宣告方式
    • var arrayName = new Array("V1","V2");
    var arrayName = new Array();
    arrayName[0]="value1";
    arrayName[1]="value2";
    var arrayName = ["V1","V2"]
  • 取陣列的值
    • array[key];

    用foreach把全部取出:

    array.forEach(function(itme, i){
    console.log(item,i);
    });
  • 陣列的屬性
    • array.property (ex. length)
  • 陣列的方法
    • array.method()
    常用的方法
    方法 說明
    sort() 排列陣列元素(小>大)
    reverse() 反轉排列陣列元素(大>小)
    pop() 減去陣列尾端元素
    push() 新增元素到陣列尾端
    shift() 減去陣列第一個元素
    unshift() 新增元素到陣列前端

  • 物件宣告方式

    • 資料的型態比較複雜、不是同一系列的話，就會用到JSON物件

    let jname = {
    name: "名字",
    age: "20",
    height: "165"
}
    物件無法直接使用.length取長度(使用 Object.keys().length)，他的key是索引(可以用字串，所以不是數字)
