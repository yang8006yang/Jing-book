認識JavaScript

一種前端的腳本語言，可以與使用者行為互動

  • 可以改HTML的內容、屬性、樣式(CSS)
  • 偵測使用者行為(點擊、滾動...)
  • 要包在 script tag 內
  • 大小寫有差

變數

宣告變數與其Scope
  • var : 看宣告位置決定是全域或區塊。(在function外宣告是全域)
  • let : 區塊變數，以{}區分，在同一{}內無法同名重複宣告
  • const : 常數，其他同let
  • 沒有以上直接賦值 : 全域變數 (小心使用)
全域變數=>window(全域物件)的屬性

資料型態

JS是個"弱型別"的語言

  • 數值Number : 可以是整數或浮點數

    • 強制取小數點位數函式：(變數).toFixed(指定位數)

  • 字串String
  • 布林Boolean
  • Undefined : 尚未被賦值
  • Null : 空值
  • Object : 上述以外的 ex.Array、Date
  • Symbol
    • 常用轉換型別的函數
    parselnt() 將字串轉為整數 從最左邊開始轉換，無法轉換則回傳NaN
    parseFloat() 將字串轉為浮點數
    Number() 將字串/物件轉為數值 無法轉換則回傳NaN
    typeof 傳回資料型別 typeof(XX) / typeof XX

運算子

    指定運算子 = += -= *= /= %= a+=3; (a=a+3)
    算數運算子 + - * / % ++ -- 加減乘除餘數
    字串運算子 + 組合字串
    布林運算式 && (x > 10)&&(y > 15) => ture/false
    0、""、false、null、undefined、NaN 都是 false
    比較運算子 == != > >= < <= === 是完全相等(資料型態也要相同)
    邏輯運算子 && || !
    a++ v.s. ++a a++ : 做運算/賦值完在加 ++a : 先加在做運算/賦值

流程控制-邏輯判斷式

  • if(...){}else{}
  • 三元運算子：條件式?成立時:不成立時
  • switch...case
    • switch(變數/運算式){
    case A:
    程式敘述;
    break;
    case B:
    程式敘述;
    break;
    .
    .
    .
    default:
    程式敘述;
}
  • 短路求值 :

    短路求值是一種邏輯運算子的求值策略。只有當第一個運算數的值無法確定邏輯運算的結果時，才對第二個運算數進行求值。(form wiki)

    || 回傳第一個 Boolean() 判斷是 true 的值
    && 回傳第一個 Boolean() 判斷是 false 的值，都是ture回傳最後一個
    ?? (空值合併運算子)回傳第一個非 null 和非 undefined 的值，都沒有回傳最後一個
