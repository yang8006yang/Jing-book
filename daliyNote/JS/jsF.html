內建函式倉庫

WINDOW
  • prompt() : 會跳出一個輸入視窗，使用者可以輸入文字，變數接收到的資料型態為文字
  • confirm() : 會跳出一個確認視窗，使用者可以點確認或取消，資料型態為布林值
STRING
  • padStart(maxLength,'fillString') : 在字串前面補字
  • padEnd(maxLength,'fillString') : 在字串後面補字
  • includes(search) : 搜尋字串，回傳布林值
  • concat() : 合併字串
  • replace("被取代","取代字") : 取代特定字，比對方式有3:/被取自/g = 全文對比、/被取自/i = 忽略大小寫、/被取自/gu = 上2效果合併
  • substring(開始位置,結束位置) : 取指定區段字串
  • split(以此分割,取多少字) : 分割字串，回傳陣列
  • trim() : 去頭尾空白
NUMBER
MATH
  • Math.abs() : 絕對值
  • Math.ceil() : 無條件進位
  • Math.floor() : 無條件捨去(不大於)
  • Math.round() : 四捨五入取整數
  • Math.trunc() : 去除小數，反回整數部分
ARRAY
  • let newArr = arr.map(?function(value, ?index){return...}) : 回傳處理後的新陣列

    • **map與foreach的差別 : map不會修改原本的陣列、有return

  • let total = arr.reduce((累計值, value)=>{return 累計值+value}) : 回傳累計後的新陣列

    • **reduce差別 : 回傳一個值

  • let newArr = arr.filter(value => ...) : 依箭頭後的條件過濾，回傳一個新陣列
