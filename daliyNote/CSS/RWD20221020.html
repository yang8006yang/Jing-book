Responsive Web Design

"content is like water" 內容像水一樣會跟著容器改變樣子


  • 媒體類型

    • all、print、screen、speech


  • Media Queries

    • 可以根據不同的媒體、可視區域、解析度等套用不同的樣子

    如下：使用link元素的media屬性設定為screen就套用
    < link rel="stylesheet" type="text/css" media="screen" href="style.css" >
    如下：使用@media，當媒體類型為print 或 screen時，套用內部內容
    @meida print, screen{
      body {font-size: 12px;)
    }
    如下：使用@import，當媒體類型為print時，套用style.css
    @import url("style.css") print;
  • 使用Queries (條件) : and(單一或同時符合)、and ,(或)、not、only

    如下：
    @media only screen and (min-width: 768px){ CSS }
    • max-width: == 這個數字以下（包含） 的適用
    • min-width == 這個數字以上（包含） 的適用
