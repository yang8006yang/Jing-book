CSS寫的位置

  • 行內CSS inline scc

    • 包在單行裡，在開頭tag後接style。 ex. tag style="color: aqua;

  • 內部CSS internal

    • 寫在html head tag裡面+style tag+{}

  • 外部css external

    • 透過外部CSS檔，在head用link tag連結
    ex.link rel="stylesheet" href="./檔名.css"

inline>internal>externak(最近的優先)

加!important; 可無視權重 直接使用(危險!不建議使用)

選擇器

  • ID 選擇器:#
  • CLASS 選擇器:.
  • 全部: *{}
  • 組合選擇器Combinators:
    descendant selector (space) 後代子元素全受影響 ex.div p { background-color: yellow; }
    child selector (>) 只有後代一層 div > p { background-color: yellow; }
    adjacent sibling selector (+) 同一層鄰近的
    general sibling selector (~) 後面全部同一層的p div ~ p { background-color: yellow; }
  • 屬性選擇器
    E[href^="http"]{} 選擇[href屬性值是以http開頭]的E元素
    E[href$="http"]{} 選擇[href屬性值是以http結尾]的E元素
    E[href*="http"]{} 選擇[href屬性值包含http的E元素

*如果有從外面copy css，自己寫的放下面(會覆蓋)
* ID > Class (當重複時ID優先)

*還可以用Pseudo
