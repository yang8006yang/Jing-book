XAMPP開發環境 安裝

xampp為一個整合型的網頁開發環境套件，裏頭已經先行整合了網頁常用的開發環境需要的設定和套件，包含了 **網頁伺服器(Apache)、資料庫(MySQL)、程式語言(PHP)、資料庫管理軟體(PhpMyAdmin)**

  1. 更換根目錄

    2. 網站的開發一般是以專案為一個單位，反應在電腦中的具體呈現方式就是資料夾目錄，因此我們會習慣一個網站的內容都放在同一個資料夾來保存維護，如果要建立新的網站或新的專案，則會開啟另外的資料夾目錄來存放

    在xampp的控制台視窗 中，點選Apache項目後的config按鈕，在彈出選單中找到Apache(httpd.conf)項目：在文字內搜尋"documentroot"修改資料夾位置

  2. 會預設開啟index.php>>可防止別人直接開啟其他檔案

流程控制

  • 選擇結構

    • if...else

      • if判斷式:如果條件式為真則執行

    • switch…case

      • switch的條件結果必須有明確的值=在多種選項中，執行符合條件式的選項

    • 三元運算子

  • 重複結構

    • FOR LOOP
      • for(設變數起始值; 條件式為真時不斷執行; 增值/減值){
      要重複執行的代碼
      }

    • While...
      • while(條件式){ 條件式為真時重複執行 }
      **do{ //要重覆執行的程式 }while(條件式);
      *和while的不同在於do..while不論條件式是否成立都會至少執行一次

    • Foreach
  • 巢狀結構

    • =迴圈+判斷式疊加(一層套一層...)

陣列

  • 空陣列宣告：

    • $a=[];

  • 陣列宣告：
    • $a[key]=value;
    • $a=[“A”,”B”,”C”]; (key從0開始)
    • $a=[1=>”A”,2=>”B”,3=>”C”]; (指定key值)
  • 取出陣列中的資料：

    • echo $a[key];

  • 多維陣列：

    • $a=[
    [1,3,4,5],
    [2,22,5,88],
    [8,2]
    ];

  • 常用函式：
    • 函式 功能
    implode("字串/符號",$arr) 以字串或符號把陣列轉為字串
    ex. $arr=['Hello','World!'] >>implode(" ",$arr);>> Hello World!
    join("字串/符號",$arr) ; 以字串或符號把陣列轉為字串(可接受參數順序對調)
    array_keys() 將key取出為一個陣列
