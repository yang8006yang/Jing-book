物件導向

以物件為主的程式開發設計方法

  • 認識OOP 
            ex.人  「Class」->屬性 : 眼睛顏色、頭髮樣式...
                   |   ->方法 : 走路方式、語調
    _______________|_______________
    |              |              |  
  John           Amy            Mary ->物件實體

    宣告範例 :

    Class People{
    public $eye='green';
    protected $hair_style="qqhair";


    public function __construct($var)
    {
        //公開行為內容
        //建構式內容
        // new建構時就執行
    }
    private function getEye(){
        return $this->eye;
        // this 指的是當下的Peopel物件 (ex.$John...)
        //私有行為內容
    }
}

$John=new People($var) 
echo $John->eye  //green
